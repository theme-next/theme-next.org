## Autoinstall Hexo & NexT & NexT Website Source on Localhost

```bash
cd /your-sites-dir
wget https://raw.githubusercontent.com/theme-next/theme-next.org/source/hexo-theme-next-autoinstall.sh
sh ./hexo-theme-next-autoinstall.sh
```
