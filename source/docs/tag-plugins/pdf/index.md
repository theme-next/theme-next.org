### Usage

```md pdf.js
{% pdf url [height] %}

[url]    : Relative path to PDF file.
[height] : Optional. Height of the PDF display element, e.g. 800px.
```

### Examples

```md
{% pdf https://example.com/sample.pdf %}
```

```md
{% pdf /path/to/your/pdf.pdf %}
```
